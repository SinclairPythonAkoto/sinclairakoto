TECHNOLOGIES

The languages/platforms used to build web apps.

Python

I use the current veriosn of Python 3.8.1 to build algorithms and create conditional logic. With Python I can also create web apps with Flask - this enables me to embed algorithms & conditional logic in my web pages, making them more dynamic and responsive. The use of Python also helps me to communicate with other languages/technologies such as HTML, SQL, Heroku & Render.

Data Storage

To store data and manage databses I use SQL (PostgreSQL). This enables me to store & validate data, and also view the stored data in other formats - like on a different page or in graphs. An example of this would be viewing the results of a feedback form.

Git

Git allows me to create opens source projects by allowing me to store and track my code through repositories. One benefit of using Git & Github is that you can collaberate easily with other developers - all working on the same project from across the globe.

Heroku/Render

Heroku & Render are free servers which allow me to deploy my web apps online. They communicate with GitHub repositories in oder to make the web app a live website. Another benifit is that you can store valuable data in environment variables and then embed it in your Python code to make it more secure - an example of this would be valuable passwords that you may need for your databases.
